윈도우객체 이용하기

브라우저(창)가 실행되면 기본적으로 생성되는 객체
한개 창마다 한개의 window객체가 생성된다.
window는 최상위객체
자식으로 BOM객체, DOM객체를 가진다.
BOM : 브라우저를 다루는 객체들 -> location,navigator, history,screen DOM : html문서를 다루는 객체 -> document, (Element)객체를 이용

새창만들기 : window.open()

[window.]open(param1,param2,param3);
1. : 창을 열었을때 연결될 url주소를 설정 2. : 창으로 여는 방법(새창,기존창을 이용) -> _self,_blank, 명칭
3. : 창에 대한 옵션(크기, 위치 ,창 메뉴설정 등)
return : 생성된 창의 window객체를 반환
