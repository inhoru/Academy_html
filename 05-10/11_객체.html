객체의 선언과 호출

js에서 객체는 key:value형식의 구조를 가진다
객체에 선언된 key는 객체에 속한 하나의 변수(속성)이다. 모든자료형값을 저장할수 있다.
key에 함수가 저장이되면 멤버메소드로 보면 된다.
key값은 중복이 불가능하다.

객체 생성하기

1. 리터럴로 생성하기 : {key : value, key : value, key : value.....}
2. 함수(생성자함수, 일반함수)를 이용해서 생성하기
- 일반함수 : 객체를 반환하는 함수
- 생성자함수 : new연산자로 생성할수있게 해주는 함수
